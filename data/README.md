## Data

This project uses the **MTSamples** dataset, a publicly available collection of de-identified medical transcription reports.

The raw data files are not included in this repository.

### Data source
The dataset can be downloaded from:
https://www.kaggle.com/datasets/tboyle10/medicaltranscriptions

### Notes
Only derived features and aggregated results are stored in this repository to comply with data usage best practices.
